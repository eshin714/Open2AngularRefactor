Angular. Two-way Data binding, it'll blow your mind.

Cliq is a single page web application that utilizes Angular's two way data binding to update data on the front end without having to refresh! With the help of Angular Material and Bootstrap, the UI is simple and effective. The page is also responsive to sizing and can be used on mobile devices! Sign-up and begin planning and chatting with friends!

NodeJS and Express. Javascript on the backend? Yes. Please.

Using Node and Express, the backend is modularized and effective. With Express middleware integrating MySQL was , we were able to create efficient AJAX requests. Also using Socket.io we integrated our Real Time Chat feature. NodeJS allows for a powerful and agile back-end using JavaScript.

MySQL . User. Events. Friends. Relational Data? Checkmate.

With our MySQL database relating our data was a breeze. With Chats, Friends, and Events we knew that we had to connect all of our data to our users and with effective queries we were able to relate our data for every request efficiently.
